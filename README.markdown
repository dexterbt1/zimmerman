zimmerman
=========

Zimmerman is a minimalist deployment tool loosely inspired by
[Shipwright](http://search.cpan.org/perldoc?Shipwright) and 
[Capistrano](http://www.capify.org/).

Zimmerman is a derived German word means `carpenter`.
